# PROJECT SPECIFICATION

**Status: FINALIZED**

## 1. Project Goal
Complete reboot of the Apostolic Theological Seminary website. We will discard the previous layout and design a completely new, modern, and high-end web application using innovative fold transitions and text animations. All content is sourced from the Yolasite archive.

## 2. Requirements & Features
- **Total Rewrite**: Discard previous iterations. Build a greenfield React/Vite application.
- **Content**: Utilize the already-scraped data from the Yolasite source (`abts_content_clean.json`).
- **Architecture**: Multiple subpages and dynamic scroll folds.
- **Innovative Animations**: Utilize scroll-triggered fold transitions and advanced text animations.
- **Gallery**: Create a dedicated gallery utilizing local images located in `public/assets`, subtitling them using their filenames.
- **Skill Integrations**:
  - `react-bits-main`: For advanced text animations, micro-interactions, and visual flair (BlurText, SplitText).
  - `taste-skill-main`: For overall design constraints, premium tactile feel, avoiding AI-slop layouts, and color consistency.
  - `ui-ux-pro-max`: For layout structure, grid systems, and responsiveness.
